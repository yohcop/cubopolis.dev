Method Detail

listenChunks

void listenChunks(chunks)
    Instruct the server that this client wants to receive updates about the given chunks. Calling listenChunks with a new list of chunks to listen overrides all previous calls to this function. The server will only send updates for the chunks passed during the last call to listenChunks.
    Parameters:
      chunks - A list of [y, x] chunk coordinates. e.g. [[0,0], [1,0]] will listen for the chunk 0,0 and 1,0

reloadChunk

void reloadChunk(<number> chunkY, <number> chunkX)
    Get the entire chunk cells from the server. This should be called whenever a non live chunk becomes live, because updates on the server are not pushed to the client when a chunk is not live. For a chunk to become live, the function listenChunks must be called.
    Parameters:
      chunkY - chunk Y coordinate.
      chunkX - chunk X coordinate.
