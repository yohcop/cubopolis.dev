getCell

string getCell(<number> chunkY, <number> chunkX, <number> cellZ, <number> cellY, <number> cellX)
    Similar to getCellFromChunk, only retrieves the chunk first. This function exists to make it easy to retrieve a cell within a chunk, but when possible (for example in drawing loops), it is more efficient to cache the currently painted chunk from getChunk, and use getCellFromChunk for each cell in the chunk.
    Parameters:
      chunkY - Y coordinate of the chunk.
      chunkX - X coordinate of the chunk.
      cellZ - Z coordinate of the cell in chunk.
      cellY - Y coordinate of the cell in chunk.
      cellX - X coordinate of the cell in chunk.
    Returns:
      cell value
